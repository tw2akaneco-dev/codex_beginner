# 全球大类资产年度复盘与展望｜YYYY

## 给我看的超短版

用不超过10条 bullet 说明：
- 今年市场最重要的主线是什么？

- 中国：政策、地产、消费、出口、人民币如何变化？
- 商品：黄金、原油、铜反映了什么？
- 风险：VIX、信用利差和地缘政治如何影响市场？

每个模块都要有“小白理解”。

- 能源和OPEC
- 贸易与关税
- 日本、欧洲、韩国
- 科技、AI或重要行业事件，如明显影响指数

每条说明：
- 发生了什么
- 为什么重要
- 影响了哪些资产
- 是否会延续到明年
- 来源和置信度

- 原油 → 通胀 → 利率 → 股债表现

每条说明：
- 年初怎么开始
- 年中如何变化
- 年末结果如何
- 明年是否继续有效
